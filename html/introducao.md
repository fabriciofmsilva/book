HTML
==========

HTML (HyperText Markup Language).

Crie um documento `index.html` e nele insira o conteúdo:

```html
Meu primeiro HTML.
```


Estrutura básica

```html
<!doctype html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Minha página</title>
    </head>
    <body>
        <!-- Conteúdo da página vai aqui -->
    </body>
</html>
```
